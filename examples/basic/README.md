# Basic Example

This example demonstrates how to create a basic Alibaba Cloud Private Zone with a single A record and VPC attachment.

## Usage

To run this example:

```bash
terraform init
terraform plan
terraform apply
```

To remove the resources:

```bash
terraform destroy
```

## Cost

Note that this example may create resources that incur costs. Run `terraform destroy` when you no longer need these resources.

## Resources Created

- 1 Private Zone
- 1 VPC attachment
- 1 A record
